2. Description

NUUC-Dispatch is an identity-verification service: it confirms a Google account's identity (email, stable account id) on behalf of a requesting NUUC application (e.g. Action Sync / GActionSheet). It does not itself manage documents, spreadsheets, or any application data.

3. Use of Google Services

NUUC-Dispatch uses Google Sign-In (GIS) with the non-sensitive openid and email scopes only, and verifies the resulting ID token server-side. It requests no access to your Drive, Docs, or Sheets.
